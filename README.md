# AI Daily Reports

Daily AI trend reports generated by Hermes.

## Layout

Daily reports are stored by date:

```text
reports/ai-daily/YYYY/MM/YYYY-MM-DD.md
```

Feature articles and Chinese translations are stored separately:

```text
features/
```

See the [feature article index](./features/README.md).

## Privacy

Before publishing, all content is sanitized to avoid committing personal information, API keys, tokens, private chat metadata, or local Hermes runtime details.
