## 社区热议

- **美团 LongCat-2.0 全量开源（官方已确认，来源见下）**：7 月 5 日以 MIT 协议开源模型权重与推理代码。1.6T 参数 MoE，每 token 动态激活约 33B–56B，原生 1M token 上下文，全程在 5 万卡国产硬件集群上从零训练（训练与推理均不依赖 NVIDIA GPU）。架构含 LongCat Sparse Attention、Zero-Compute Experts、按任务门控的 MOPD 路由（Agent / Reasoning / Interaction 三组专家）。基准：SWE-bench Pro 59.5（高于 GPT-5.5 的 58.6）、Terminal-Bench 2.1 70.8、SWE-bench Multilingual 77.3、BrowseComp 79.9。上线前曾以匿名代号「Owl Alpha」在 OpenRouter 冲上调用量前列。原生兼容 Claude Code、OpenClaw、Hermes Agent。来源：testingcatalog、stretchcloud

- **Qwen-RobotNav（社区传闻/未确认）**：社区转述称 Qwen 团队推出 2B–8B 统一模型族，覆盖视觉语言导航（VLN）、ObjectNav、目标跟踪与自动驾驶，可零样本部署到真实四足机器人。Qwen 官号过去 24 小时无对应发文，标注未确认。来源：HuggingPapers

- **OpenAI AI 手机传闻（社区传闻/未确认）**：社区流传 OpenAI 正加速一款以持续后台 AI Agent 为核心、不运行传统 App 的智能手机，量产目标 2027 上半年。无官方确认。来源：lesliedqbb

- **OpenClaw 生态安全实践**：HF 维护者 @onusoz 发布 `uvx hf-auth-helper agent login`，用受限 token 权限（read + discussion.write）让 OpenClaw / Hermes 等 agent 安全创建 PR，规避 repo 删除或 force-push 风险。来源：onusoz

## 总体趋势观察

- **中国开源模型继续主导 agentic 编程赛道**：LongCat-2.0 在 SWE-bench Pro 上超过 GPT-5.5，且以 MIT 协议放出，延续了国产开源模型「高参数 MoE + 全国产算力 + 完全可下载」的路线，进一步压缩闭源前沿模型在实用编程任务上的差距。

- **官方静默、社区喧闹的错位**：过去 24 小时美西各大官号（OpenAI、Anthropic、Google DeepMind 等）几乎无实质发布，真正的信息增量来自第三方开源发布与开发者实践，说明当前一线厂商的节奏更多体现在生态与工具层，而非频繁的官方公告。

- **Agent 生态从「模型」转向「环境与工具治理」**：OpenClaw 的受限 token 安全实践、多模型编排（研究用一款、执行用一款、敏感任务走本地）等讨论表明，社区焦点正从「哪个模型更强」转向「如何安全、可控地让 agent 落地运行」。
