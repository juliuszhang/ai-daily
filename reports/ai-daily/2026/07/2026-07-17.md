## 社区热议

### 1. 🏆 Moonshot 发布 Kimi K3：世界最大开源模型，开源与闭源差距"刚刚消失"

Moonshot AI（月之暗面）正式发布 **Kimi K3**，社区震动。核心参数：

| 项目 | 数据 |
|------|------|
| 参数规模 | **2.8 万亿参数**（2.8T），世界首个开源 3T 级模型 |
| 架构 | 稀疏 MoE + **Kimi Delta Attention (KDA)** + **Attention Residuals (AttnRes)** |
| 上下文 | **100 万 token** 原生上下文窗口 |
| 视觉 | 原生多模态视觉支持 |
| 智能指数 | **Artificial Analysis Intelligence Index 第 3 名**，与 Anthropic Opus 4.8、OpenAI GPT-5.5 并列 |
| 编码基准 | 追平 GPT-5.5 |
| 速度提升 | KDA 在百万 token 上下文中解码速度提升 **6.3 倍** |
| 训练效率 | AttnRes 训练效率提升约 **25%**（仅增加 <2% 成本） |
| 定位 | 长周期编码、知识工作、深度推理 |

社区评价两极分化但总体正面：
- **开源派欢呼**：Glitchwire 称"开源与闭源的差距刚刚消失"；trendingtopics.eu 称"中国震撼，开源权重模型被提升到前沿水平"
- **实力对比**：Fortune 称 K3 将中国 AI 推入"Fable 级别"；Axios 称开源模型首次达到前沿水平
- **客观差距**：Moonshot 自认整体性能仍逊于 Claude Fable 5 和 GPT-5.6 Sol，但在其评测套件中 K3 持续优于其他被测模型
- **价格信号**：The Decoder 指出 K3 接近 GPT-5.6 Sol 和 Fable 5 水平，同时标志着"超低价中国 AI 时代"的终结

来源：[Moonshot 官博](https://kimi.com/blog/kimi-k3)、[MarkTechPost](https://marktechpost.com)、[Axios](https://axios.com)、[Fortune](https://fortune.com)、[TechCrunch](https://techcrunch.com)、[Glitchwire](https://glitchwire.com)、[SiliconANGLE](https://siliconangle.com)

### 2. Mira Murati 离开 OpenAI 后首发模型，完全开源

OpenAI 前 CTO Mira Murati 创办的新公司发布了首款 AI 模型，采用**完全开源**策略。社区热议这是迄今离开 OpenAI 的最高级别高管首次推出独立产品，且选择开源路线与 OpenAI 闭源路线形成鲜明对比。来源：[Decrypt](https://decrypt.com)

### 3. xAI 开源 Grok Build 编码 Agent 引发社区热议

xAI 将 Grok Build 以 Apache 2.0 开源后，社区讨论集中于：可运行任意模型（本地推理/开源模型/托管模型）、支持完全本地执行、默认零数据保留。部分开发者关注其 bug bounty 计划。社区评价与此前 SpaceXAI 零数据保留政策一脉相承。来源：[MarkTechPost](https://marktechpost.com)、[x.ai](https://x.ai)

### 4. Google Gemini 3.5 Pro 延迟，Alphabet 股价下跌

Bloomberg 和 CNBC 报道 Google Gemini 3.5 Pro 因未达到内部目标而推迟发布，Alphabet 股价应声下跌。社区讨论 Google 在 AI 竞赛中的节奏问题，部分开发者认为 Google 反复跳票正在消磨市场信心。来源：[Bloomberg](https://bloomberg.com)、[CNBC](https://cnbc.com)

### 5. Google DeepMind 推进 AI 生物安全研究

### 6. Nous Research 融资 7500 万美元，估值 15 亿

### 9. 美国初创企业转向中国便宜模型

NPR 报道越来越多美国初创企业因成本考虑转向使用 GLM、DeepSeek、Kimi 等中国模型。社区中 Coinbase 工程师迁移案例持续发酵，有开发者报告同等任务成本降低 50-80%。来源：[NPR](https://npr.org)

### 10. LM Studio 发布 Bionic：面向开源模型的 AI Agent 应用

LM Studio 推出 Bionic，一款面向开源模型的 AI Agent 桌面应用，支持本地运行多种开源模型并具备 Agent 工作流能力。社区评价为"本地版 Claude Code"的有力竞争者。来源：[9to5Mac](https://9to5mac.com)

## 总体趋势观察

1. **🏆 Kimi K3 改写开源格局**：2.8 万亿参数、智能指数第 3、编码追平 GPT-5.5——Moonshot 用一个模型宣告开源与闭源的差距已实质性消失。这是中国 AI 从"低成本替代"正式进入"前沿竞争者"行列的标志性事件。

2. **开源成为本周主旋律**：Kimi K3 开源、Mira Murati 首款模型开源、xAI 开源 Grok Build——从前 OpenAI 高管到中国创业公司再到 xAI，开源路线形成合力，与 OpenAI/Anthropic 闭源路线分庭抗礼。

3. **AI 安全从讨论走向制度建设**：Google DeepMind 推进 AI 生物安全研究并呼吁国际标准体系、OpenAI 发布 GPT-Red 红队工具、Anthropic 推动各州立法——三家头部公司不约而同将 AI 安全推向制度层面。

4. **Agent 框架赛道资本热度攀升**：Nous Research 15 亿估值、LM Studio Bionic、GitHub Copilot SDK——开源/本地 Agent 工具正在成为新的投资和创业焦点。

5. **Google 节奏焦虑加剧**：Gemini 3.5 Pro 延迟导致股价下跌，与 Kimi K3、Grok Build 开源等密集发布形成鲜明对比，社区对 Google AI 进度的信心持续动摇。
