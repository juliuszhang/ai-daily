## 社区热议

- GLM-5.2（~750B MoE / ~40B 激活，MIT 开源，1M 上下文）在 front-end 编码 Code Arena 排名靠前，漏洞发现部分指标追平 Opus 4.8 / GPT-5.5，推理成本约为对标模型的 1/4–1/6；David Sacks、Marc Andreessen 等公开点赞。GLM-5.2 / DeepSeek / MiniMax 被开发者称为「mini DeepSeek moment」。
- Base MCP 让 ChatGPT / Claude / Cursor / Codex 直连 Base 链钱包，可查余额、swap、借贷，私钥不出钱包、每笔交易需用户确认。
- ZCode 发布 GLM-5.2 免费桌面 agent 环境；GitHub Copilot 加入 Kimi K2.7 Code（首个进入选择器的开源权重模型）；Cursor 推出原生 iOS app；阿里开源 Page Agent（将 DOM 转为文本地图的浏览器自动化库）。
- 微软启动「Frontier Company」，投入 $2.5B、将约 6000 名工程师嵌入企业客户；扎克伯格向员工表示 AI agent 进展慢于预期。
