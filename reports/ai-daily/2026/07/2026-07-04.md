- Base MCP 是本轮讨论度最高的 agent 工具之一：让 ChatGPT / Claude / Cursor / Codex 直连 Base 链钱包（查余额、swap、借贷），私钥不出钱包、每笔交易需用户确认。
- 中国开源冲击波：GLM-5.2 / DeepSeek / MiniMax 以 1/4–1/6 成本逼近美国前沿模型，西方开发者兴趣快速上升，引发对美国技术领先的政策讨论。

- 大厂信号：微软启动「Frontier Company」（$2.5B，把 6000 名工程师嵌入企业客户）；扎克伯格据称向员工承认 AI agent 进展慢于预期。

## 总体趋势观察

1. Agent 化进入深水区：竞争焦点从「模型分数」转向长程稳定性、工具使用、上下文管理，以及「未知项发现」等人机协作方法论。
2. 科研 AI 成新战场：Claude Science 把 agent 从写代码推向真实湿实验室工作流，是本轮最具想象力的方向。
3. 开源性价比持续挤压闭源：中国三强（智谱/DeepSeek/MiniMax）在真实使用榜的存在感越来越强，多模型编排、BYOK 成为开发者共识。
