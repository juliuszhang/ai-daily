## 社区热议

- **Gemini 开始直接生成 Google Slides 演示文稿**：Google Workspace 官方展示了在 Slides 中根据用户的 Docs、Sheets 和 PDF 生成完整演示文稿的功能，文字、图片及版式均可继续编辑。

- **Claude Security 成为开发者讨论量较高的新功能**：官方发布帖披露的功能包括提交前差异扫描和全代码库扫描；该帖在统计时已获得约 **69 万次浏览、9,100 次点赞和5,800 次收藏**。

- **Grok Build 的游戏开发演示被集中传播**：Elon Musk 转发并推荐 Grok Build，其中一个演示使用 Grok 4.5、Unity CLI 与 MCP 在单次会话中搭建可运行的 macOS 3D 岛屿探索游戏；另一组演示展示了代码、素材、世界和玩法生成。

- **Open Teach 发布为开源、自托管的技能录制工具**：开发者展示了通过录制屏幕和语音，将操作过程转换成可复用智能体技能的方案，并表示可配合不同模型及智能体使用。

- **Relay 展示多模型代码路由工作流**：该开源技能采用“前沿模型规划—快速模型执行—前沿模型复核”的流程，并接入 Codex、Cursor 和 Claude Code。

- **Kimi K3 获 OpenAI 高管公开评价**：据 Bloomberg 报道，OpenAI 总裁 Greg Brockman 称 Moonshot AI 的 Kimi K3“具有竞争力”，同时表示目前无法确定其是否使用了 OpenAI 模型蒸馏技术。

1. **编码智能体的竞争重点正在转向安全与工程化。** Claude Code 将漏洞扫描嵌入终端和提交前流程，社区则在补充技能录制、多模型路由以及完整应用生成能力。

2. **企业智能体继续从“回答问题”走向“执行操作”。** OpenAI Presence 明确覆盖企业系统调用、获批动作和人工升级，产品设计重点已转向权限、流程与责任边界。

3. **Gemini 向科研和办公场景同步扩展。** DeepMind 通过算力与模型额度支持科学研究，Google Workspace 则把可编辑演示文稿生成直接嵌入 Slides。

4. **本窗口内没有新的旗舰基础模型正式发布。** 官方动态主要集中在智能体部署、代码安全、数据连接器和现有模型的场景化落地。
